# 学習記録・可視化AIアプリ  

## 概要  

学習ログCSVをアップロードすると、学習時間や理解度を可視化・分析し、LLMを用いて学習改善提案を生成するStreamlitアプリです。  

日付、科目、学習時間、理解度を含むCSVを読み込み、pandasで集計したうえで、matplotlibで学習時間の推移を可視化します。また、理解度と平均学習時間をもとに、学習上の課題を「努力不足」「理解不足」「問題なし」に分類し、改善アクションを提示します。  

さらに、集計結果をLLMに渡すことで、学習傾向、課題、科目別の改善案、次の1週間の行動計画を自然文で生成します。  

## 開発背景  

自分の学習時間や理解度を客観的に振り返るために開発しました。    
単に勉強時間を記録するだけでなく、学習時間と理解度を組み合わせて分析し、次に取るべき行動まで提示できるようにすることを目的としました。

## 主な機能  

### 1. CSVアップロード機能  

学習ログCSVをアップロードし、アプリ上で表形式で確認できます。  

必要なCSV列は以下です。  

| 列名 | 内容 |  
|---|---|  
| date | 学習日 |  
| subject | 学習科目 |  
| study_time | 学習時間 |  
| understanding | 理解度 |  

### 2. CSV列チェック  

必要な列が不足しているCSVをアップロードした場合、エラーを表示し、後続処理に進まないようにしています。  

### 3. ルールベース分析  

理解度と平均学習時間をもとに、学習記録を以下の3種類に分類します。  

| 条件 | 問題タイプ |  
|---|---|  
| 理解度が低く、学習時間が平均未満 | 努力不足 |  
| 理解度が低く、学習時間が平均以上 | 理解不足 |  
| 理解度が中・高 | 問題なし |  

分類結果に応じて、以下のような推薦アクションを表示します。  

| 問題タイプ | 推薦アクション |  
|---|---|  
| 努力不足 | 学習時間を増やしましょう |  
| 理解不足 | 別教材や復習を検討しましょう |  
| 問題なし | この調子で継続しましょう |  

### 4. 学習時間の可視化  

以下の2種類のグラフを表示します。  

- 日付ごとの総学習時間の推移  
- 科目別総学習時間の推移  

### 5. LLMによる学習改善提案  

CSVデータをそのままLLMに渡すのではなく、pandasで以下の情報に要約してからLLMに渡しています。  

- 総学習時間  
- 学習日数  
- 平均理解度  
- 科目別集計  
- 問題タイプ別件数  
- 直近の学習記録  

これにより、入力データを整理し、LLMが学習傾向を把握しやすい形で改善提案を生成できるようにしています。

## 実行方法  

### 1. リポジトリをクローン  

```bash
git clone リポジトリURL
cd study-analystics-ai-app
```

### 2. 必要なライブラリをインストール  
pip install -r requirements.txt  

### 3. APIキーを設定  
".streamlit/secrets.toml"を作成し、以下のようにOpenAPIキーを設定します。  
OPENAI_API_KEY = "your_openai_api_key_here"  

### 4. アプリを起動  
python -m streamlit run app.py  

## 工夫した点  
CSVをそのままLLMに渡すのではなく、pandasで要約してから渡す設計にした。  
学習時間と理解度を組み合わせ、課題を「努力不足」「理解不足」「問題なし」に分類した。  
可視化だけでなく、改善アクションや次週の行動計画まで提示できるようにした。  
APIキーをコードに直接書かず、Streamlitのsecretsで管理した。  
CSVの列チェックを追加し、不正な入力に対してエラー表示できるようにした。
